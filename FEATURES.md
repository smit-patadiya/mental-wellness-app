# MindMitra — Feature Spec (MVP + researched differentiators)

> The full feature set, re-grounded in [FRAMEWORK.md](FRAMEWORK.md) + [SOLUTION.md](SOLUTION.md). Visual = `public/design-variant.html` (8 screens). Build owners in [ANTIGRAVITY.md](ANTIGRAVITY.md).

## Core (built ✅)
- **One input** — feeling chip · voice · text (no forms).
- **AI analysis** — reflection + insight + **technique** + encouragement (Gemini, key server-side).
- **6 rotating techniques** — breathing · reframe (CBT) · grounding · self-compassion · break (DBT) · affect-labeling. *(Fixes "always breathing": per-technique icon + real variety.)*
- **Crisis safety** — deterministic guard → helplines (tested).
- **Mood trend + history** — local-first dashboard.
- **Voice journaling**, **dark mode**, **local-first privacy**.

## Differentiators (building now)
| Feature | Screen | Why (research) |
|---------|--------|----------------|
| **Hidden-trigger Insights** | Insights | the HERO — patterns across entries ("dips after evening mocks + low sleep") that mood-sliders miss |
| **Exam countdown context** | You | advice adapts as the exam date nears (NEET/JEE…) |
| **"What helps you"** | You | closed-loop — learns which technique works best for *this* student |
| **Sound therapy** | Sounds | ambient music that shifts by mood |
| **Color therapy** | global | palette shifts by mood + time of day |
| **Weekly reflection** | Insights | short AI narrative of the week |
| **Welcome / onboarding** | Welcome | calm first-run, graphic-first |
| **Streak-free progress** | You | gentle, no guilt (stressed students) |

## Therapies applied (human-psychology grounding)
CBT (reframe) · mindfulness + breathing/meditation · self-compassion (Neff) · DBT distraction (break) · affect-labeling · **color therapy** · **sound therapy**. See [PSYCHOLOGY.md](PSYCHOLOGY.md).

## Navigation
Bottom bar (pinned bottom): **Today · Insights · Breathe (center FAB) · Sounds · You**.

## Design language
Graphic-first, **low-text**, animated breathing orb, soft **blurred backgrounds**, calm sage + mood palettes, crafted SVG icons (no emoji), gentle motion (reduced-motion safe), desktop = mobile device frame.
